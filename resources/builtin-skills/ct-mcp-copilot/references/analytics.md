# CipherTalk MCP Analytics Playbook

## Tool selection

### `get_global_statistics`

Use for:

- 总消息量
- 发送/接收比例
- 首次/最后一次消息时间
- 总体活跃天数

### `get_contact_rankings`

Use for:

- 聊得最多的是谁
- 某时间段联系最频繁的人
- Top N 联系人

### `get_activity_distribution`

Use for:

- 活跃时段
- 星期分布
- 月度分布

## Recommended order

1. Confirm whether the user wants overall scope or a specific scope
2. If overall, start with one analytics tool that best matches the question
3. Combine tools only when the user wants a broader summary

## Battle report

- “战报：问题是总览，先看全局统计。”
- “战报：问题是聊得最多的人，直接拉联系人排行。”
- “战报：问题是作息规律，开始看活跃时段分布。”
